# Product Management System 🛒

A simple **Product Management System** built using **HTML, CSS, and JavaScript**.  
This project simulates a basic e-commerce cart system where users can add, edit, and remove products.

## 🚀 Features

-  Add products to cart
-  Edit product details (name, price, quantity)
-  Remove products from cart
-  View all added products in cart
-  Data persistence using localStorage
-  Responsive and simple UI

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
